选择布局方案

当前问题：max-width: 1280px + margin: 0 auto 导致宽屏下内容区居中但 Sidebar 固定左侧，视觉上偏右。下面是 3 种修复方案：

A

全流式布局 推荐

移除 max-width 限制，内容始终撑满可用区域，加水平 padding 防止贴边。1440px、1920px 均全屏。

1920px 宽屏效果示意
.content-wrap {
  max-width: 1280px;
  margin: 0 auto;
  padding: 24px 32px;
  width: 100%;
}
B

宽 max-width 居中

将上限放宽到 1600px，主流 1440px / 1920px 基本全屏，4K 超宽屏下仍有居中保护。

1920px 宽屏效果示意（约 90% 宽度）
.content-wrap {
  max-width: 1280px;
  max-width: 1600px;
  margin: 0 auto;
  padding: 24px 32px;
  width: 100%;
}
C

各页面自控宽度

AppLayout 完全去掉限制，各页面自己决定内容宽度。Chat 全宽，Dashboard 可独立配置卡片布局。灵活但需要逐页处理。

布局结构示意
各 View 自定义宽度
/* AppLayout 去掉全局限制 */
.content-wrap { width: 100%; height: 100%; }

/* 各 View 按需加 */
.page-inner { max-width: 960px; margin: 0 auto; }
